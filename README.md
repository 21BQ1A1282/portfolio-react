## 🚀 Getting Started

### Prerequisites
- Node.js
- npm or yarn

### Installation
1. Clone the repo:
   ```bash
   git clone https://github.com/21BQ1A1282/portfolio-react.git
   ```

2. Navigate to the directory:
   ```bash
   cd portfolio
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev 
   ```
